# SYNOPSIS

## Title of the Project:
**CLOUD-BASED MOVIE TICKET BOOKING & MANAGEMENT SYSTEM WITH AI CHATBOT ASSISTANT**

## Introduction:
The Cloud-Based Movie Ticket Booking & Management System with AI Chatbot Assistant is a comprehensive web application designed to revolutionize the movie ticket booking experience. This system integrates advanced technologies including artificial intelligence, real-time seat selection, and cloud-based infrastructure to provide a seamless and user-friendly platform for movie enthusiasts and theater administrators.

The traditional method of booking movie tickets often involves standing in long queues, limited seat visibility, and time-consuming processes. This project addresses these challenges by providing an online platform with an intelligent chatbot that guides users through the entire booking process conversationally, making ticket booking as simple as having a conversation.

## Objectives:
The primary objectives of this project are:

1. **AI-Powered Assistance**: To develop an interactive AI-powered chatbot with animated robot interface for conversational ticket booking
2. **Real-Time Visualization**: To implement real-time seat selection with visual 3D cinema-style representation
3. **Secure Transactions**: To integrate multiple payment gateways (UPI, Card, Wallet) for secure and flexible transactions
4. **Comprehensive Management**: To provide a comprehensive admin panel for theater, movie, and booking management
5. **Cloud Architecture**: To create a scalable cloud-based architecture supporting concurrent users
6. **Enhanced UX**: To enhance user experience with animated cyan robot assistant and intuitive interfaces
7. **Complete Lifecycle**: To manage the complete booking lifecycle from selection to e-ticket generation

## Scope of the Project:
The system covers the entire lifecycle of movie ticket booking and management:

**For Users:**
- Browse movies by genre, date, and rating

- Visual seat map with 3D screen representation

**For Administrators:**

**Technical Scope:**

### 2.2.2 SYSTEM ARCHITECTURE

The system follows a three-tier architecture:

1. Presentation Layer (Frontend):
   - React.js single-page application
   - Component-based architecture
   - Responsive UI with CSS3 animations
   - Animated Robot component
   - Context API for state management

2. Business Logic Layer (Backend):
   - Node.js with Express.js
   - RESTful API design
   - JWT middleware for authentication
   - Input validation middleware
   - Chatbot controller

3. Data Layer (Database):
   - MongoDB NoSQL database
   - Collections: Users, Movies, Theaters, Showtimes, Bookings, Snacks
   - Mongoose ODM for data modeling
   - Indexing for performance

Communication Flow:
User Interface (React) <-> REST API (Express.js) <-> MongoDB Database

Key Components:
1. Animated Robot - Independent React component with 8 states
2. Chatbot Engine - Conversation state management
3. Seat Map - Visual component with real-time updates
4. Payment Gateway - Integrated payment processing
5. Admin Dashboard - Comprehensive management interface

const createBooking = async (req, res) => {
  const { showtimeId, seats, snacks, paymentMethod } = req.body;
  const userId = req.user._id;
  
  const showtime = await Showtime.findById(showtimeId)
    .populate('movie theater');

    showtime.bookedSeats.includes(+"${s.row}"+)
  );
  if (unavailable.length > 0) {
    return res.status(400).json({ message: 'Seats unavailable' });
  }
  
  // Calculate amounts
  const seatsAmount = seats.reduce((sum, s) => sum + s.price, 0);
  const snacksAmount = snacks.reduce((sum, s) => sum + (s.price * s.quantity), 0);
  const totalAmount = seatsAmount + snacksAmount;

    showtime: showtimeId,
    movie: showtime.movie._id,
    theater: showtime.theater._id,

    snacksAmount,
    totalAmount,
    showDate: showtime.date,
    showTime: showtime.time,
    paymentMethod,
    paymentStatus: 'completed',
    bookingStatus: 'confirmed'
  });
  
  await booking.save();
  
  // Update showtime
  showtime.availableSeats -= seats.length;
  showtime.bookedSeats.push(...seats.map(s => +"${s.row}"+));

### C.1 Backend - Authentication Controller

`javascript
// authController.js
const User = require('../models/User');
const bcrypt = require('bcryptjs');
const generateToken = require('../utils/generateToken');

// @desc    Register new user
// @route   POST /api/auth/register
// @access  Public
const register = async (req, res) => {
  try {
    const { name, email, password } = req.body;

    // Check if user already exists
    const userExists = await User.findOne({ email });
    if (userExists) {
      return res.status(400).json({ 
        success: false, 
        message: 'User already exists' 
      });
    }

    // Hash password
    const salt = await bcrypt.genSalt(10);
    const hashedPassword = await bcrypt.hash(password, salt);

    // Create user
    const user = await User.create({
      name,
      email,
      password: hashedPassword,
      role: 'user'
    });

    if (user) {
      res.status(201).json({
        success: true,
        message: 'User registered successfully',
        data: {
          id: user._id,
          name: user.name,
          email: user.email,
          role: user.role,
          token: generateToken(user._id)
        }
      });
    }
  } catch (error) {
    res.status(500).json({ 
      success: false, 
      message: error.message 
    });
  }
};

// @desc    Login user
// @route   POST /api/auth/login
// @access  Public
const login = async (req, res) => {
  try {
    const { email, password } = req.body;

    // Check user existence
    const user = await User.findOne({ email });
    if (!user) {
      return res.status(401).json({ 
        success: false, 
        message: 'Invalid credentials' 
      });
    }

    // Verify password
    const isMatch = await bcrypt.compare(password, user.password);
    if (!isMatch) {
      return res.status(401).json({ 
        success: false, 
        message: 'Invalid credentials' 
      });
    }

    res.json({
      success: true,
      message: 'Login successful',
      data: {
        id: user._id,
        name: user.name,
        email: user.email,
        role: user.role,
        token: generateToken(user._id)
      }
    });
  } catch (error) {
    res.status(500).json({ 
      success: false, 
      message: error.message 
    });
  }
};

module.exports = { register, login };
`

        snacks: selectedSnacks
      });

      const botMessage = {
        type: 'bot',
        text: res.data.reply + '\n\nRedirecting to payment...',
        timestamp: new Date()
      };
      setMessages(prev => [...prev, botMessage]);

      // Store booking data
      const bookingData = {
        showtime: selectedShowtime,
        seats: selectedSeats,
        snacks: selectedSnacks,
        fromChatbot: true
      };
      sessionStorage.setItem('chatbotBooking', JSON.stringify(bookingData));

      // Redirect to payment page
      setTimeout(() => navigate('/payment'), 1500);
    } catch (error) {
      console.error(error);
    }
  };

  return (
    <div className="chatbot-container">
      <div className="chatbot-header">
        <AnimatedRobot action={robotAction} size="small" />
        <h2>AI Movie Booking Assistant</h2>
      </div>
      
      <div className="messages-container">
        {messages.map((msg, idx) => (
          <div key={idx} className={+"message "+}>
            {msg.type === 'bot' && (
              <AnimatedRobot action={msg.robotAction || 'idle'} size="small" />
            )}
            <div className="message-bubble">{msg.text}</div>
          </div>
        ))}
      </div>
    </div>
  );
};
`
